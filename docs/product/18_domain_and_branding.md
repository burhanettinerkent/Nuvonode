# Domain and Branding

## Final brand decision

Use `Nuvonode` as the main brand and `Nuvonode Mesh` as the open-source project name.

The user owns:

```text
nuvonode.com
```

## Brand positioning

Nuvonode is a community-powered AI inference network.

The name should communicate:

- node-based architecture
- community provider network
- practical AI model inference
- internal credit economy
- open-source infrastructure
- room to grow beyond only GPUs

## Product naming

Use these names consistently:

```text
Brand: Nuvonode
Open-source project: Nuvonode Mesh
Main repository: nuvonode
Provider binary: nuvonode-provider
CLI binary: nuvonode
Dashboard: Nuvonode App
API: Nuvonode API
Credits: Nuvonode Credits
Router: Nuvonode Model Router
```

## Naming rules

Use:

- Nuvonode
- Nuvonode Mesh
- Nuvonode Provider
- Nuvonode Credits
- Nuvonode Model Router
- Community Provider
- Trusted Provider
- Managed Provider

Do not use:

- Nuvonode Coin
- Nuvonode Mining
- Nuvonode Cash
- GPU mining
- passive income guarantee
- decentralized supercomputer for 70B+ in V1

The project must not create crypto/mining expectations. V1 credits are internal platform usage credits only. They cannot be withdrawn, sold, transferred, or converted to fiat money.

## One-line description

Nuvonode Mesh lets developers call open AI models through one API while community GPU providers earn internal credits by serving approved inference jobs.
